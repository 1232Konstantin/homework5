Detailed Description

Класс Document имитирует координатную плоскость с расположенными на ней графическими юнитами

Реализует связь с фабрикой объектов и хранилищем данных. Сам в свою очередь предназначен для использования во viewer

Выполняет функции контроллера в модели MVC
