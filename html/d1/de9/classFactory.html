Detailed Description

Класс Factory это фабрика объектов для созданиия разных вариаций графических юнитов

Представляет собой синглтон (указатель на фабрику доступен по вызову статической функции getInstance() )

Всем поддерживаемым типам юнитов соответствует определенный индекс
