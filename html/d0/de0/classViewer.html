Detailed Description

Класс Viewer предназначен для организации взаимодействия с пользователем

Выполняет отображение юнитов на экране, масштабирует изображение, перемещает окно просмотра, обеспечивает интеррактивное взаимодействие с пользователем по средством контекстных меню (класс меню не показан)

Класс должен взаимодействовать с обработчиком мыши посредством вызова соответствующих функций

Класс может быть расширен библиотекой элементов, поддерживающей Drag and Drop (класс библиотеки не показан)
